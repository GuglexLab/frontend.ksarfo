Modern CSS Techniques

This is my take on @keyframers' Ocean Overlays codepen exercise. It uses CSS only to achieve masked text, navigation, and animation.

There are 4 hidden radio inputs on this page that keep track of the current "scene". The radios inputs are associated with labels that are also hidden.

The masked text can be achieved by setting a background image to the tag containing the text, then by setting background-clip: text.

The primary layout technique used here is CSS grid. The parent defines grid-template-* and the children use grid-row and grid-column to determine where the element will span on the grid.
